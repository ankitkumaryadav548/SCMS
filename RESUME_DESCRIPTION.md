## 📝 Resume Bullet Points (Action-Oriented)

### Option 1: Full-Stack & System Design Focused
- **Architected a 3-tier Smart City Management System** combining React 18, Node.js Express, Java Spring Boot 3, and MongoDB to optimize city traffic, emergency dispatches, and parking bookings.
- **Engineered a Java DSA Microservice** running **Dijkstra** and **A\*** pathfinding algorithms to achieve **sub-millisecond (<0.5ms)** route optimizations for city navigation and emergency priority routing.
- **Built a Real-Time Traffic Simulation Engine** using **Socket.io WebSockets**, pushing live 2-second congestion metrics and automated bottleneck alerts for 17 city road segments.
- **Implemented an array-backed FIFO Queue data structure** ($O(1)$ enqueue) in Node.js to manage real-time parking slot reservations and citizen service request queues.
- **Designed a utility network layout optimizer** leveraging **Kruskal's Minimum Spanning Tree (MST)** with Disjoint Set Union-Find to minimize municipal infrastructure piping/cabling costs.

### Option 2: Software Engineering & Backend Heavy
- **Developed scalable REST APIs and WebSocket servers** with Express.js and Mongoose, enforcing Role-Based Access Control (RBAC) across Administrator and Citizen user portals.
- **Integrated Java Spring Boot microservices with Node.js** via HTTP REST proxies to benchmark pathfinding latency between Dijkstra's algorithm and A\* heuristics.
- **Optimized MongoDB database schema** with indexed geospatial attributes and automatic Mongoose document seeding upon deployment.
- **Deployed containerized services to cloud environments**: Frontend on Vercel, Backend API on Render, Java Microservice on Railway, and Database on MongoDB Atlas.
